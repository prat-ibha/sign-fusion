# Finger-Speak-AI-Listen

AI-powered Sign Language Translation toolkit using React and Three.js.

## Project Structure
- `src/Animations`: Contains alphabet and word animations for sign language.
- `src/Components`: Reusable UI components.
- `src/Pages`: Different sections of the application (Home, Convert, Learn, etc.).
- `src/Models`: 3D models (xbot, ybot) for sign language animation.

## Features
- Real-time text-to-sign language translation.
- Speech-to-sign language translation using AI.
- Learning modules for Indian Sign Language (ISL).
- Community video creation and sharing.

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start the project: `npm start`.
